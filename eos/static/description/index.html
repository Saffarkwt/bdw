# End of Service (EOS) Module The End of Service (EOS) module is a comprehensive solution for managing and calculating end-of-service benefits for employees according to Kuwaiti labor law. This module provides an easy mechanism for calculating end-of-service benefits based on the reason for termination and the employee's length of service. ## Key Features - End-of-service benefit calculation according to Kuwaiti labor law - Support for different termination reasons with specific calculation methods - Configurable calculation formulas based on service periods - Complete workflow for EOS requests (draft, submitted, approved, HR approved, paid) - Integration with HR and accounting modules - Comprehensive reporting and analysis tools - Multi-company support - Security access control with dedicated user groups - Multilingual support (English and Arabic) ## Screenshots ![EOS Request Form](screenshot_eos_request.png) ![EOS Calculation Wizard](screenshot_eos_calculation.png) ![EOS Dashboard](screenshot_eos_dashboard.png) ## Documentation For detailed information, please refer to the following documentation: - [User Guide (English)](../../doc/user_guide_en.md) - [User Guide (Arabic)](../../doc/user_guide_ar.md) - [Installation Guide (English)](../../doc/installation_guide_en.md) - [Installation Guide (Arabic)](../../doc/installation_guide_ar.md) ## Support For support or customization requests, please contact the development team. ## License This module is licensed under LGPL-3. ## Author Developed by Manus AI
